# PHYSICS-ENGINE

This is a very basic physics engine with the sole purpose of racking up a few base hours for Hackclub Flavortown.
I used this as an opportunity to set up GitHUb properly and practice using it correctly.

The code allows you to place blocks and watch them experience gravity until they collide with the bottom of the screen. 
You can spawn blocks that have a random chance of being either red or blue. You can pick up blocks and drop them.

Using A and D you can move the blocks left and right, just not individually.

# Known Errors:
